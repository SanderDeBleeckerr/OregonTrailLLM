# The game master runs on PyTorch (text_server.py), so torch + the packages

# install torch FIRST from PyTorch's own index, then this file:
#
#   pip install torch --index-url https://download.pytorch.org/whl/cu128
#   pip install -r requirements.txt
#
# cu128 covers everything from GTX 10-series through RTX 50-series (Blackwell
# needs it — cu124 and older wheels have no sm_120 kernels). AMD, Apple
# Silicon, or CPU-only: pick your command at
# https://pytorch.org/get-started/locally/ (bitsandbytes NF4 is CUDA-only;
# text_server falls back to bf16/fp32 elsewhere).
#
# Tested with torch 2.7+cu128 on Python 3.13.
